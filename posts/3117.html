Hexo新建文章
Hexo
发布日期:   2022-07-01
更新日期:   2024-01-14
文章字数:   627

简化版本

  1. hexo new postname 新建文章
  2. 在postname.md文件里编写文章内容
  3. 本地预览无问题后上传
    • hexo c 清除本地缓存
    • hexo g 生成新的blog内容
    • hexo s 本地预览效果，ctrl+C退出预览
    • hexo d 上传到自己的网站
详细与注意事项

一. 新建hexo文章

hexo new postname 必须在blog目录下执行
（包括后面的所有hexo相关命令都要在blog所在目录下执行）

执行后，会在blog文件的source/post的目录下
生成一个名称为postname的md文件
如果之前装的有image的插件，则还会有一个同名的空文件夹，用于放置文章的配图.

postname就是新建文章的名称
名称可以是英文数字中文或者三者的随意组合，但是不能有空格，可以用-连接

二. 编写文章内容

在postname.md文件中，开头部分，是文章的配置信息

文章Front-matter介绍

    ---
    title: 第一篇文章    #文章标题
    date: 2022-07-02 18:23:25   #文章创建日期
    author: JiuHaohao           #可以单独在某一篇署名作者，空白的话就默认配置文件里的作者信息
    img: /source/_posts/hello-world/1.jpg #文章封面
    summary: 这是你自定义的文章摘要内容，如果这个属性有值，文章卡片摘要就显示这段文字，否则程序会自动截取文章的部分内容作为摘要
    categories: markdown   #文章的分类
    tags: 
        - markdown
        - 日记    #文章的标签，可以有多个
    ---

注意

  1. 文章封面也可以用外链（推荐用图床），或者自定义文件内的图片，没有的话就是默认特征相册里的图片
    封面用外链的话本地测试的时候加载很慢
    不写的话就是默认本地特征相册里的,本文用的就是外链
img: 'https://w.wallhaven.cc/full/k7/wallhaven-k7q9m7.png'

其他的

还有top,toc,password,cover,coverimg,reprintPolicy,mathjax,详细参考下面表格

在开头部分的下面，即可用markdown语法编辑自己的文章内容，
md语法教程可参考我的“Markdown基本语法”这一篇博文

三. 本地预览和上传

hexo clean
清除本地上一次生成的blog内容缓存
hexo g
生成新的blog内容
hexo s
本地运行 检查效果

ctrl+鼠标左键点击链接 即可预览
ctrl+C 退出预览
hexo d
一键上传到自己的网站
