三 插入emoji表情

四 代码块

    ```(代码的语言标签)
     代码内容
    ```

1.只有表明了代码语言标签，才会有相应的高亮显示;如果不标明语言可能会出现一些显示bug，比如

    ```
     代码内容
    ```

如果里面换成拼音也是一样的bug,所以就是不标明语言标签的话，就会出现这种问题
（此外代码块的语言标签中，markdown可以缩写为md;不对，缩写后没有高亮了,全是白色的，代表不认识这种语言，甚至必须是markdown，m不能大写，否则也没有高亮;尽管在md文件里是高亮的，但是上传到网页后是白色的，所以这个高亮很可能是md文件自带的，于是你以为md也可以）
2.代码块嵌套（就是代码块套代码块）的时候要注意层级关系，可以用tab键调整一下结构，否则可能会错乱;

示例

    ```css
    .gitment-container .gitment-header-like-btn {
    color: #ab47bc;
    }  
    ```

效果

.gitment-container .gitment-header-like-btn {
    color: #ab47bc;
}
